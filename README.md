# 🌄 travel2 — 我的旅行网站

一个简洁、快速、可部署的个人旅行展示站，由 [爪爪 (Claw)](https://github.com/openclaw) 为你搭建。

## ✨ 特点
- 纯 HTML/CSS，无 JS 依赖
- 响应式设计，手机/桌面均友好
- 图片本地存储，加载快、不依赖外链
- 中文支持完善（UTF-8 + 微软雅黑 fallback）

## 🖼️ 当前图片
- `images/zhangjiajie_1.jpg` — 张家界奇峰（来自 Hugo 主题示例）
- `images/scenic_view.jpg` — 风景截图（你上传的 screenshot.png）

> 💡 你可以随时替换 `images/` 中的图片为自己的旅行照片（建议命名：`place_1.jpg`, `place_2.jpg` 等，避免中文/空格）

## 🚀 部署到 GitHub Pages
1. 进入仓库设置 → Pages
2. 分支选 `main`，目录选 `/root`
3. 保存 → 等待几分钟，访问 `https://revivalzs.github.io/travel2`
